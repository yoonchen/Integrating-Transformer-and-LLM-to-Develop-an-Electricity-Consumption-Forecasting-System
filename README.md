The following Jupyter Notebooks are provided for reproducing the experiments:

- `llama_training.ipynb`: This notebook contains the training process for the Llama-based language model, detailing the data preprocessing, training steps, and evaluation metrics.
- `BERTscore.ipynb`: This notebook demonstrates how BERTScore is used to evaluate the similarity between generated text and reference text in energy-saving recommendations.
- `V2_CAG.ipynb`: This notebook implements the CAG (Content-based Attention Generator) model to generate recommendations based on the predicted energy consumption.
